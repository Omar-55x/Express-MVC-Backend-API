# Express MVC Backend API

A backend API built with Express.js following the MVC architecture. I have bulit and still bulding on this server to learn and strengthen backend development concepts.

## Features & Concepts Practiced (ongoing)

- Routing | Implemented and learned how to manage different kinds routes
- Middleware | Implemented built-in, third-party and custom middleware (including request logging and error handling)
- MVC | Organized the code in Model-View-Controller pattern for better scalability and maintainability
- REST | Built a REST API endpoints for performing CRUD operations across application resources
- API Testing | Tested APIs using tools like Thunder Client to ensure functionality
- MongoDB | Integrated MongoDB through Mongoose to model data schemas
